# Robot Lab 0 — Simple Wheeled Robot (MicroPython)

Goal: make the robot move with the simplest possible code.

## Files
- `src/motor.py` - motor control helpers (forward/reverse/left/right/stop)
- `src/main.py` - simple test routine to prove wiring & directions
- `docs/wiring.md` - pin map and wiring notes

## Phase 0 Success Criteria
- Motors spin the correct direction
- Robot can: forward / reverse / left / right / stop
- No WiFi, no UI, no state machine, no acceleration logic
